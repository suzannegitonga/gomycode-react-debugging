# React Debugging Checkpoint

## Project Description
This project is a sample React application created for a debugging checkpoint. The goal was to use React Developer Tools to identify and fix issues related to component state and props.

## Features
- Component tree inspection using React Developer Tools.
- State management debugging (fixing direct state mutation).
- Props passing debugging (fixing mismatched prop names).

## How to Run
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser to the local host link provided in the terminal.

## Tools Used
- React
- Vite
- React Developer Tools (Browser Extension)
- Git & GitHub
